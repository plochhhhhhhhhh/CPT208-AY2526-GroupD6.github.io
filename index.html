Project Overview

We design a smart campus exploration system that transforms traditional map-reading into an intelligent storytelling journey.

It reduces cognitive load and guides users naturally through location-aware interaction.

Core Features

📍 Smart Trigger

Stories activate automatically near buildings (15m range)

🧭 Dual Navigation

Choose curated routes or AI-optimized paths

🧠 Story Mode

Learn campus culture through immersive audio

📱 Camera Guidance

Visual arrows and building information overlay

UI Showcase

User Journey (Luna — Exchange Student)

Persona: Luna, 20, exchange student from Germany

First-time visitor unfamiliar with XJTLU campus

Context: First time visiting campus, unfamiliar with layout.

Goal: Explore key buildings efficiently while learning interesting stories.

Frustration: Overwhelmed by maps and language barriers.

START

Opens the campus app and views the home screen.

Taps "Get Started" but feels unsure where to begin.

Pain Points: "Where do I start?" "Which building is important?"

Emotion: Confused

NAVIGATE

Selects a curated tour and activates camera guidance.

Follows directional hints while walking.

Pain Points: "I can't read maps" "Are the directions clear enough?"

Emotion: Hesitant

DISCOVER

Approaches a building and automatically enters a trigger zone.

Audio storytelling begins, sharing cultural insights.

Pain Points: "Will it auto-play?" "Audio timing unclear"

Emotion: Surprised

ACHIEVE

Completes all stops and finishes the tour.

Sees progress reach 100% and receives a reward badge.

Pain Points: Too many options before finishing, prefers automation.

Emotion: Accomplished
